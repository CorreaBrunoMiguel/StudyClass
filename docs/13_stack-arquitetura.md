## Frontend

- Next.js (React + App Router)
- Tailwind CSS

## Backend

- NestJS (Node + TypeScript)
- REST API

## Banco

- Postgres via Docker Compose (local)
- Prisma ORM
- Conteúdo das unidades: JSONB (decisão já registrada no modelo de dados)
