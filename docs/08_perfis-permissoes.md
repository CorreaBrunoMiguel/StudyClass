# Perfis e Permissões (v1)

docs/08_perfis-permissoes.md

## Perfis

- ALUNO
- ADMIN

## ALUNO

- Criar conta / login (somente se e-mail estiver na whitelist)
- Matricular-se em cursos permitidos
- Consumir unidades (conteúdo + interações + provas)
- Ver notas, tentativas, histórico
- Ver e baixar certificados/badges

## ADMIN

- Gerenciar whitelist de e-mails
- CRUD de cursos
- CRUD de módulos (ID global)
- CRUD de capítulos/subcapítulos
- CRUD de conteúdo JSON (blocos)
- CRUD de provas
- Visualizar resultados/tentativas
